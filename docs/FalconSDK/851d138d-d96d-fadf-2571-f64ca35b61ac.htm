PropertyDescription Properties
The [PropertyDescription](891585f6-6578-38bb-73fd-4e40b70a1423.htm) type exposes the following members. Properties
  Name Description
Instance
Gets the occurrence index of the interface object. The first occurrence of a specific object type has Instance=1.
MainDPT
Gets the main DPT number
MaxSize
Gets the maximum number of array elements.
ObjectIndex
Gets the index of the interface object if known. This is only valid if Instance == 0.
ObjectType
Gets the type of the interface object if known. This is only valid if Instance > 0.
PropertyDataType
Gets the type of the property data, see also PropertyDataType.
PropertyId
Gets the Id of the interface object property.
PropertyIndex
Gets the index of the interface object property.
ReadRights
Gets the property read access rights.
SubDPT
Gets the sub DPT number
WriteEnabled
Gets a value indicating whether the KNX object is [write enabled].
WriteRights
Gets the property write access rights.
