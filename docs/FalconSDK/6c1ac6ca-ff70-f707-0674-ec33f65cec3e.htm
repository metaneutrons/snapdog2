GroupAddresses
In the case of this being a Group Access token.

This contains the group address this access token applies to. The Master secret will be the group key
Id
An access token id. It is used for update and deleting an item on a device. The id is unique on a device and should be unique in an installation.
KeyConfiguration
Confirmation key. For example with a symmetric proof of-possession key [RFC 8392]

  1. "coap_dtls" for X.509 certificates [RFC 8392] or,
  2. "coap_oscore" for OSCORE
Scope
In the case of this being a Tool Access token.

Access Scope. A list of interfaces that define the access scope. As defined in [RFC 8392]

The master secret will be the tool key.
