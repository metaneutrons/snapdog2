  Member name Value Description
Connected 0 The connection to the bus interface is established and working and the interface reports a working connection to the medium.
Closed 1 The connection to the bus interface is still closed or has been closed.
Broken 2 The connection to the bus interface has been successfully opened, but there is a (temporary) communication problem. Falcon is trying to reestablish the communication in the background. If the reconnect succeeded, the status returns to Connected.
MediumFailure 3 The connection to the bus interface is established and working, but the interface reports a failed connection to the medium.
