IoTSoftwareUpdatePackage Class
A software update package that can be directly pushed to an IoT device. The [TransferChunkPolicy](039cf006-ef0b-459f-a42a-22b96694ef4e.htm) species the specifics of how the binary data shall be streamed to the device.
