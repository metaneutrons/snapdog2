KNX Falcon SDK

KNX Application-Layer Data
Falcon SDK provides a rich set of conversion facilities for most of the standardized KNX datapoint types and property data types. Conversion Within the Falcon API, group values are represented as [GroupValue](cfefd12c-88f9-89e7-b457-f55df2d98718.htm) objects, and interface object property values simply as byte arrays. The semantics of these values is given implicitly by the datapoint type (DPT) / property data type (PDT) meta data. The conversion facilities in the [Knx.Falcon.ApplicationData](0568debf-ec6f-5a30-5f9c-95b44ae4374c.htm) namespace allow co easily convert this internal representation to data types more suitable for representing the values. The implementation relies of the data provided in the knx\_master.xml file (in schema version 2.3) which therefore must be present in either %commonappdata%/KNX/XML/project-23 (%localappdata% usually maps to C:\ProgramData on Windows and /usr/share on Linux and macOS) or the application base directory. If both are present, the one with the higher version is used. #### Datapoint types (DPT) A rich set of DPT converters (classes derived from [DptBase](815b65b3-3f60-ad3b-5e2d-334360e73f2d.htm)) allow to convert between [GroupValue](cfefd12c-88f9-89e7-b457-f55df2d98718.htm) and the canonical data type suitable for representing the semantic meaning. The converter may either be instantiated directly (e.g. as new Dpt6(), or better by using one of the factory methods in [DptFactory](094b4984-5cba-dd2f-23d9-2ac3c96d5729.htm). #### Property data types (PDT) A rich set of PDT converters (classes derived from [PdtBase](f234e59f-8a32-153a-47ab-39626909c11b.htm)) allow to convert between the raw byte array and the canonical data type suitable for representing the semantic meaning. The converter may either be instantiated directly (e.g. as new PdtDouble(), or better by using one of the factory methods in [PdtFactory](2119053a-fbdc-b1be-7f72-a46e2291e10d.htm). Generic editing helpers The .NET framework supports implementation of generic editors by the ![External Link](../media/ExternalLink.png)TypeDescriptor infrastructure. The [KnxEditableValue](611e2410-d141-3d27-fa1d-5a0c3312debd.htm) class is a wrapper class cooperating with the [DPT converters](#DPT) providing this feature.
