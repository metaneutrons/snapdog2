KNX Group Communication
Falcon SDK allows to communicate with the KNX installation using group messages. Each runtime function in KNX is identified by its [GroupAddress](0ffb286f-6b36-38cf-1840-0b7b53633047.htm). There are three services (Write\_Group\_Value, Read\_Group\_Value, Respond\_Group\_Value) using group addresses which are all supported by Falcon both for sending and receiving. Optional section title Add one or more sections with content Security Add one or more sections with content Group Values On the bus, group values are raw, untyped bit streams. KNX messages represent short values (up to 6 bits) in a different, more compact format. For interoperability, different datapoint types (DPT) are defined in the KNX Handbook (chapter "3/7/2 Datapoint Types"). Falcon contains [DPT Converters](fb3cf115-c09e-4dc4-8782-69654a7ac5e3.htm#DPT) to easily translate between raw bits and types more suitable for handling in applications. RF Multi If Falcon is connected to the installation via an RF Multi USB interface, the necessary information about channels and acknowledge slots needs to be configured.
