Knx.Falcon.Configuration Namespace
This namespace contains the classes used for configuring bus access, i.e. the ConnectorParameters class ans its derived classes. Classes
  Class Description
ConnectorParameters
This is the base class for all connector specific configuration parameters.
EiblibConnectorParameters
This class provides parameters used to open a bus connection via a device supporting the legacy Eiblib protocol.
IoTConnectorParameters
This class provides parameters used to open a bus connection via the KNX IoT Point API.
IpDeviceManagementConnectorParameters
This class provides parameters used to access a KNXnet/IP device directly via a Local Transport Layer connection (Device Management v2) or local MProp services (Device Management v1).
IpRoutingConnectorParameters
This class provides parameters used to open a bus connection via the KNXnet/IP Routing backbone.
IpTunnelingConnectorParameters
This class provides parameters used to open a bus connection via a KNXnet/IP Tunneling interface.
IpUnicastConnectorParameters
This class is the base class for KNXnet/IP connector parameters used to open a KNXnet/IP tunneling or device management connection.
UsbConnectorParameters
This class provides parameters used to open a bus connection via a KNX USB interface.
