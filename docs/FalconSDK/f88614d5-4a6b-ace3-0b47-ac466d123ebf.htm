IoTOperationPolicy Class
A policy that determines how a function is performed, and in particular what happens if the function can for some reason not be performed. This entails specifying how many times the operation shall be attempted in total. this includes a [NumberOfInnerAttempts](eb68a136-6911-91f3-0605-211a9ad58f48.htm) that determines how many times supplementary operations shall be performed. e.g. operations such as determining the device IPv6 address and port given its serial Number. Timeouts are also specified. Note that for some operations both minimal and maximal timeouts are necessary. - [MaximumTimeout](0c9c8c78-9801-226b-676e-df6436679e2d.htm) - Determines when to abort waiting to see if a operation was successful and treating the operation as a failure. - [MinimumDuration](2d9e6845-15a2-1671-2301-dd22e0a8e1d8.htm) - Determines for operations that are open ended for how long at the very least waiting for some events to occur shall be done. ActiveWaitingMaximumTimeout
The time after which the active waiting is aborted. During this time period every ActiveWaitingRefreshDelay the current state is polled.
ActiveWaitingRefreshDelay
This time is the interval between retries when actively waiting e.g. for a state change.
MaximumTimeout
The given time after which any individual operation is canceled. Take note that some operations consist of multiple steps so this timeout will be used for each step.
MinimumDuration
This timespan takes effect on operations such as discovery that wait for results to arrive for a set timespan.
NumberOfAttempts
The number of attempts determines the total number of attempts performed. This number shall always be at least 1.
NumberOfInnerAttempts
The number of attempts determines the number of attempts performed for operations that constitutes a partial step within the completion of another operation. An example would be an operation for reading a group objectTable, this value then determines how many attempts may be performed for reading each group value. This number shall always be at least 1.
