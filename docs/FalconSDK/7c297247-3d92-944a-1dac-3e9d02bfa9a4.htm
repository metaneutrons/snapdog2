NoOperation 0 Does nothing.
StartLoading 1 Indicates that the device will start being loaded from now.
LoadComplete 2 Indicates that the device has been loaded completely.
Unload 4 Resets the device table data so that a new application program can subsequently be loaded to the device.
Remarks The loading sequence will roughly be as follows: 1. Set the load state machine to Unload to reset any existing data. 2. Set the load state machine to StartLoading to indicate that configuration will now be loaded. 3. Download configuration onto the device. 4. Set the load state machine to LoadComplete to indicate that the configuration is loaded and that the device can now be used.
