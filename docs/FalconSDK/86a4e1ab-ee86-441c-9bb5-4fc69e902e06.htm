KNX Network Management
Falcon SDK allows to perform KNX network management operations. The network management methods are located in the [KnxNetwork](e64cdf86-a7cc-51aa-3acf-342bb005b0c5.htm) class. Device Discovery A basic functionality is to check if a specific individual address is reachable on the KNX network using the [PingIndividualAddressAsync](5c2a3646-80e0-9812-adc9-4f729e3f943c.htm) method.
