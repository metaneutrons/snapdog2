IIoTSecurityExtensions Methods
The [IIoTSecurityExtensions](cd52490c-21a6-61b3-bb99-2d43cbc7e3d2.htm) type exposes the following members. Methods
  Name Description
AuthenticateDeviceAsync
Use a password to establish a temporary secure connection that will then be used to write the given Tool Key to the IoT Device.

After calling this method other methods can be used to configure the IoT device securely with messages encrypted using the provided tool key.
ReadAccessTokenItemAsync
Determines a IoTAccessToken with a specific access token id from a device.
ReadAuthenticationTableAsync
Determines the authentication table of a device.
WriteAuthenticationTableAsync
Sends a request to a device, requesting to alter the authentication table.
