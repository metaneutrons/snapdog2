Remarks The following formats are accepted: - an empty string results in [None](36b9a66e-f46f-a24c-4c33-853b01947d90.htm). - a decimal or hexadecimal (with "$" or "0x" prepended) number results in a [PL(UInt16)](631f6bcf-95a1-1936-dd83-43ce17b154a3.htm) domain address. - a string in the form "xxxx:xxxxxxxx" (where each x represents a hexadecimal digit) results in a [RF(UInt64)](be93e6d7-50be-aeb9-bd89-4666f0ec4978.htm) domain address. - an IP address in dotted notation results in an [IP(IPAddress)](c348bb9b-1f94-49c4-8310-0ea0ebf026d0.htm) domain address.
