[Top](#PageHeader) Remarks A KnxBus object uses a specific method to connect to the KNX bus. The method and its details are configured either via a [ConnectorParameters](a00d43d8-ca1f-e911-2f70-bea37d855086.htm) object or via a connection string; see [Configure Bus Connections](b8e64132-30a6-4311-95f5-0f27ab53f8b0.htm) for details. Constructing and connecting is a two-step process. This allows to fine-tune some settings and to subscribe to events before actually connecting to the bus. When no longer needed, use the [DisposeAsync](92e532d3-a6d7-b715-206f-a193d302cd19.htm) method to free resources held by the object or - equivalently - wrap into an await using statement (C# version 8 or higher). Using [Dispose](83bd104d-9840-bbd6-a78c-79579a8a79ab.htm) or - equivalently - a using statement is also possible, but should be avoided as it might block for some time. Examples C# VB C++ F# Copy // C# 8 or higher await using (var bus = new KnxBus("Type=USB")) { await bus.ConnectAsync(CancellationToken.None); // more... } // C# 7 or less (using the Async helper methods) await Async.Using(new KnxBus("Type=USB"), async bus => { await bus.ConnectAsync(CancellationToken.None); // more... }); // C# 7 or less (using try ... finally) var bus1 = new KnxBus("Type=USB"); try { await bus1.ConnectAsync(CancellationToken.None); } finally { await bus1.DisposeAsync(); } ' Using the Async helper methods Await Async.Using(New KnxBus("Type=USB"), Async Function(bus) Await bus.ConnectAsync(CancellationToken.None) ' more...
