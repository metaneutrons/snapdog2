GroupValue Properties
The [GroupValue](cfefd12c-88f9-89e7-b457-f55df2d98718.htm) type exposes the following members. Properties
  Name Description
IsShort
Returns if SizeInBit is less than 7, i.e. if the group value is transmitted in the first (partial) ASDU octet.
SizeInBit
Gets the size in bit from the group value, e.g. for a one bit group value 1 is returned.
TypedValue
Gets the specified typed value, e.g. for a one bit value a boolean is returned.
Value
Gets the group value as byte array.
