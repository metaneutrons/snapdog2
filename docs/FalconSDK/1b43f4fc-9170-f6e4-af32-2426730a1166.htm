ConnectorParameters ToConnectionString Method
Overload List
  Name Description
ToConnectionString
Returns a string representation of the ConnectorParameters object. This can be converted back to a parameter object using FromConnectionString(String).
ToConnectionString(SecureString)
Returns a string representation of the ConnectorParameters object where the security relevant parameters are included encrypted with a password. This can be converted back to a parameter object using FromConnectionString(String, SecureString).
