Use Falcon from Powershell
It is possible to use Falcon SDK from a ![External Link](../media/ExternalLink.png)PowerShell script. With the current Powershell versions, the following Falcon features require special attention: - There is no direct support for cleaning up of disposable objects, so you need to call Dispose() manually in a finally. - There is no support for asynchronous methods, so you need to use options 2 or 3 in [Use Falcon from non-async code](2c3cb26b-0759-4796-93a6-6ad269585682.htm) to circumvent this. Example The following example shows how to connect to the bus and do some group communication PowerShell Copy $bus = [Knx.Falcon.ManufacturerSdk.KnxBus]::New("Type=USB") try { Write-Host "Connecting" $bus.Connect() # Send the 1-bit group value '1' to group address 1/1/1 Write-Host "Sending group value 1 to 1/1/1" $bus.WriteGroupValue("1/1/1", $true) # Register an event handler so we get informed if some group message is received Write-Host "Listening to group messages" Register-ObjectEvent -InputObject $bus -EventName GroupMessageReceived -SourceIdentifier "GroupMessageReceived" | Out-Null while ($true) { $event = Wait-Event -SourceIdentifier "GroupMessageReceived" if ($event.SourceArgs.Value -ne $null) { Write-Host $EventArgs.EventType, $EventArgs.DestinationAddress, $EventArgs.Value } } } finally { Write-Host "Cleaning up" Unregister-Event -SourceIdentifier "GroupMessageReceived" $bus.Dispose() } Copyright Â© 2010-2024 KNX Association cvba, Brussels, Belgium
