IObjectRepresentationData
A generic intermediary object representation format that serializes arbitrary object into a common format.

This common object representation format can either be conveniently created via an initializer directly or can be converted to and from JSON (Javascript Object Notation) as well as CBOR (Concise binary Object representation)

The following types are available: An ObjectRepresentationMap is a map of key value pairs that allows to model named properties. An ObjectRepresentationArray is a list of values that allows to model a collection of data in a given order. An ObjectRepresentationValue is a single item which allows single values to also be represented.
