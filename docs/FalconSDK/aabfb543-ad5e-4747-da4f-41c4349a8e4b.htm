IoTPublisher Class
Entry in the Publisher table of a KNX IoT device. The publisher table fulfills the purpose of determining from which endpoints to receive group communication The required Information is: - [Id](57dfe23f-a1b1-46e2-9888-2adf3e59eeb9.htm) - ID of entry unique with respect to the table - [GroupAddresses](9ceef224-24f7-e3e4-8716-f57ef96667c7.htm) - Group Addresses for which to message the specified recipient And either of these can then be used to uniquely identify the publisher endpoint: - The [Url](2a22f535-40d9-2b37-65ce-9c49e3fe4200.htm) - Defines a url including IPv6 address and port to tell the device from which publishers to receive. - The [IndividualAddress](8bfcf431-54ae-3eb4-c144-c8b5f1fb09b8.htm) and [InstallationId](000eef7a-d83c-e742-0b7d-ebabdb6d9069.htm) - Define the individual address and installation id of the target device which the device can use to determine IPv6 address and port.
