Connector Type Data
IpRoutingConnectorParameters
  • MulticastAddress: If set in the parameter object to a value not equal to DefaultMulticastAddress, the value is compared to the value in the keyring file; on mismatch, the method returns and the parameter object is left unchanged. Otherwise, the value is set to the value from the keyring file.

IpTunnelingConnectorParameters At least the property IndividualAddress must be set before calling this method to determine the tunneling slot to be used.
  • UserId: If set in the parameter object to a value greater than 1, the security data for the specified user is loaded; if the UserId is not configured for this tunneling slot, the method returns and the parameter object is left unchanged. Otherwise, the value is set to the lowest UserId from the keyring file configured for this tunneling slot.
  • UserPasswordHash
  • DeviceAuthenticationCodeHash
IpDeviceManagementConnectorParameters At least the property HostAddress must be set before calling this method. The method will read the device description so the device must be reachable via the network.
