KNX Falcon SDK

Knx.Falcon.Sdk Namespace
This namespace contains the main class of the KNX Falcon SDK - KnxBus - as well as the classes used for device and network management - KnxDevice and KnxNetwork. Classes
  Class Description
KnxBus
Represents a specific connection to the KNX bus
KnxDevice
Provides access to KNX device management procedures
KnxNetwork
Provides access to KNX network management procedures
