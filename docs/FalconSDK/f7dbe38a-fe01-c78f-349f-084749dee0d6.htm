InitialValue 0 Indicates no result being ready yet

The device returns this result right after a download or upgrade has been initiated.

Meaning that this indicates that a download / upgrade is currently still in progress.
SoftwareUpdatedSuccessfully 1 Indicate the software having been upgraded successfully.
NotEnoughFlashMemoryForSoftwarePackage 2 Indicates that during the software update the device ran out of flash memory for the software package.
OutOfRamDuringDownload 3 Indicates that during the software update the device ran out of RAM during the download.
ConnectionLostDuringDownload 4 Indicates that the connection the the server was lost during the package download.
IntegrityCheckFailureForSoftwarePackage 5 Indicates that the device refused to perform an upgrade due to it failing the integrity check.
UnsupportedPackageType 6 Indicates that the manufacturer specific package type was incompatible with the device or is currently not supported.
InvalidUrl 7 Indicates that the url provided for finding a software update was invalid i.e. the server could not be reached.
SoftwareUpdateFailed 8 Indicates that the upgrading of the software failed due to an unspecified reason.
UnsupportedProtocol 9 Indicates that the device has been instructed to use a protocol for downloading the software update package that it does not support.
