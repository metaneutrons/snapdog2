DptCompoundGeneric Class
This class is responsible for encoding and decoding compound datapoint types for which no explicit implementation is provided. The raw value is translated to and from [KnxCompoundValue](f56c57b9-ad49-5cc8-4caa-8666c1e34cf9.htm).
