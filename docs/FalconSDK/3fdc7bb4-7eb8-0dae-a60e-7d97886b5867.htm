IoTRecipient Properties
The [IoTRecipient](9fbd0be6-055a-bff6-7c39-f594ab865bc1.htm) type exposes the following members. Properties
  Name Description
GroupAddresses
A list of group addresses which shall be sent to the configured recipient.
GroupId
Id
ID of the Function Point Recipient Table Entry.
Remarks
This ID shall be unique with respect to the table.
IndividualAddress
The individual address identifying the endpoint of the recipient.
InstallationId
KNX Installation ID of the recipient.
MaximumRetransmissions
Defines the amount of allowed retransmissions for sending group messages.

If this property is omitted then the following values are used:

  • For not confirmed messages: 0 retransmissions
  • For confirmed messages: 4 retransmissions
NonConfirmable
Defines if group messages shall be confirmed.

Per default this is set to false indicating that group messages shall be confirmed.

If this is true then non-confirmable requests will be sent. Retransmissions will not be possible in this case. (see also: MaximumRetransmissions)
Path
The resource path of the recipient used for sending group messages.
Url
The url identifying the endpoint of the recipient.
