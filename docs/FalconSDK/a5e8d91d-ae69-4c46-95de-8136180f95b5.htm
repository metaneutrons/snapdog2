Discover USB Interfaces
The [UsbDeviceDiscovery](878be22e-9b20-50f2-3287-9d5dedf1827a.htm) class allows to discover the currently attached USB devices. ### Discover attached USB interfaces 1. Call [GetAttachedUsbDevices](feded26f-c529-fd31-0102-ad0c60eb19d8.htm) to enumerate USB devices. 2. The enumerated [UsbDeviceDiscoveryResult](e3caec7e-c518-4e30-d834-39b8d1495fc7.htm) objects can be used to display a list in the UI. It is also possible to create a [UsbConnectorParameters](e5991e63-a99c-66cc-c91f-b3c025475d42.htm) object or the equivalent connection string from it. Example C# VB C++ F# Copy foreach (var usb in KnxBus.GetAttachedUsbDevices()) { Console.WriteLine(usb); // convert to a connection string var connectionString = usb.ToConnectionString(); Console.WriteLine(connectionString); // or directly to a USBConnectorParameters object var connectorParameter = UsbConnectorParameters.FromDiscovery(usb); } For Each usb In KnxBus.GetAttachedUsbDevices() Console.WriteLine(usb) ' convert to a connection string Dim connectionString = usb.ToConnectionString() Console.WriteLine(connectionString) ' or directly to a USBConnectorParameters object Dim connectorParameter = UsbConnectorParameters.FromDiscovery(usb) Next No code example is currently available or this language may not be supported.
