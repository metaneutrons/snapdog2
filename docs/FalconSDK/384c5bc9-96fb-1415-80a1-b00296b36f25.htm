Remarks If this is not , Falcon connects via the given Cloud Connect API. In addition, the following properties mus be set: - [CloudConnectorId](2e2a4351-a51f-c2f9-b07a-7a1d45b5030c.htm) must contain the connector ID - [CloudAuthorization](5f47cea4-2fdc-6f11-2792-ae0536250336.htm) must contain the access key or access token - [SerialNumber](842688bc-84b5-515c-7fcc-713b341bb4e0.htm) (optional) may contain the serial number of the KNXnet/IP server in the installation to connect to NOTE: Only for test purposes, currently http is supported.
