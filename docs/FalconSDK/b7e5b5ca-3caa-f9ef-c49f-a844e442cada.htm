Knx.Falcon.ApplicationData Namespaces
The Knx.Falcon.ApplicationData assembly provides various conversion facilities for KNX datapoint types (DPT) and KNX property data types (PDT). Namespaces
Namespace Description
Knx.Falcon.ApplicationData
This namespace contains types for repesenting KNX-specific data
Knx.Falcon.ApplicationData.DatapointTypes
This namespace contains conversion facilities from and to KNX datapoint types, as used e.g. in group communication.
Knx.Falcon.ApplicationData.Editing
This namespace contains helper classes allowing generic editng of (datapoint) values in the UI.
Knx.Falcon.ApplicationData.IoT
Knx.Falcon.ApplicationData.MasterData
This namespace contains datapoint type information from the KNX master data
Knx.Falcon.ApplicationData.PropertyDataTypes
This namespace contains conversion facilities from and to KNX property data types.
