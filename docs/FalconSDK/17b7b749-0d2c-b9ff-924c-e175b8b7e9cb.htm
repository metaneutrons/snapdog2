IIoTDevice Properties
The [IIoTDevice](7ae1ff5c-5ee9-13bb-d217-66a14aa99a1f.htm) type exposes the following members. Properties
  Name Description
Security
Provides access to the various security related operations.

In particular it allows to bootstrap a new device via a password authenticated key exchange and allows to configure authentication table data. This means it becomes possible to add Tool keys and group keys.
SerialNumber
This represents the unique serial number that identifies the IoTDevice.
SleepCycle
Sleep cycle if the device is a sleepy device.
SoftwareUpdate
Provides access to the various security related operations.

In particular it allows streaming a software update package to as device directly (PUSH) or requesting the device to download a package from a specified software update server (PULL)
